# Flutter docs attestation

1. Flutter's site describes Flutter as an open-source framework for building natively compiled multi-platform applications from a single codebase.
2. Flutter's site says Flutter is powered by Dart, a language optimized for fast apps on any platform.
3. Flutter API docs/search results state Flutter has a large package ecosystem on pub.dev, including packages for hardware services like camera, widgets, animations, and integrations.
4. Flutter platform-channel docs state Flutter can communicate with platform-specific code asynchronously via `MethodChannel`, with platform implementations on Android and iOS; plugin packages connect Dart APIs to platform-specific implementations.
5. Flutter state-restoration docs describe `RestorationManager` and restoration buckets as a framework facility for restoring application state after the OS kills an app.
