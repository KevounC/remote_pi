# fast-check docs attestation

1. fast-check documentation describes fast-check as a property-based testing library for JavaScript and TypeScript.
2. It generates inputs, runs assertions over many cases, and shrinks failures to the smallest input that still reproduces the problem.
3. The getting-started docs distinguish synchronous `fc.property` from asynchronous `fc.asyncProperty`.
4. The Vitest tutorial says fast-check can be used with test runners and recommends `@fast-check/vitest` for Vitest integration.
