# React Native docs attestation

1. React Native's New Architecture page says the new architecture has been available experimentally since 0.68 and is being made/defaulted across the ecosystem.
2. The 2024 New Architecture announcement says React Native 0.76 ships with the New Architecture by default; it adds support for modern React features and type-safe native module/native component systems without the legacy bridge.
3. React Native 0.86 release notes (June 2026) describe continued release activity and Android 15 edge-to-edge / DevTools improvements with no user-facing breaking changes.
4. The Strict TypeScript API docs say stronger TypeScript types are available from React Native 0.80 onward, but opting in is a one-time breaking change.
5. React Native AppState docs say AppState reports foreground/background/inactive states; the WebSocket docs describe the global WebSocket class as the web-standard class, with the page still work-in-progress.
