# Tauri mobile docs attestation

1. Tauri v2 docs describe mobile development commands such as `tauri android dev` and `tauri ios dev`.
2. Tauri mobile plugin docs say plugins can run native mobile code written in Kotlin/Java and Swift, with Android and Swift plugin templates.
3. Tauri mobile docs indicate Rust/mobile integration exists, but native mobile plugins still require platform-specific code for mobile APIs.
