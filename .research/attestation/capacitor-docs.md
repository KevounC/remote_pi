# Capacitor docs attestation

1. Capacitor's README describes Capacitor as a cross-platform API and code execution layer that calls Native SDKs from web code and supports writing custom native plugins.
2. Capacitor also provides first-class Progressive Web App support, allowing one app to deploy to app stores and mobile web.
3. Capacitor Background Runner docs describe an event-based standalone JavaScript environment for executing JavaScript outside the web view for handled events.
4. The source set found in this pass did not establish Capacitor as a turnkey solution for reliable continuous mobile WebSockets under iOS/Android background restrictions.
