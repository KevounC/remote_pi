# Alloy 6 attestation

1. Alloy documentation describes the Analyzer as the tool that checks specs by converting a model into a SAT formula.
2. Alloy command docs distinguish `run` for finding satisfying examples and `check` for finding counterexamples to assertions within a given scope.
3. Alloy sets/relations docs describe Alloy expressions as sets of atoms and relations.
4. The Alloy 6 page states Alloy 6 adds temporal features and a `var` keyword for mutable signatures/fields; non-`var` signatures/fields are static across time.
5. Alloy time docs state Alloy 6 added temporal operators, making dynamic-system modeling easier; bounded steps are used when checking temporal models.
