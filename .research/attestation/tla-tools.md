# TLA+ tools attestation

1. The TLA+ tools page describes tool changes since *Specifying Systems* and points to current versions/tool documentation.
2. The TLA+ high-level view describes TLA+ as a language for modeling software above the code level and hardware above circuit level; the most commonly used engineering tool is the TLC model checker, with a proof checker also available.
3. The TLA+ Toolbox page describes the Toolbox as an IDE for TLA+ tools, including running the TLC model checker and exploring TLC error traces.
4. The TLC wiki result describes TLC as an explicit-state model checker for finite models of real-world systems.
5. The PlusCal manual search result identifies PlusCal as a TLA+ algorithm language with a 2024 C-syntax manual and a section on checking algorithms.
