# `go_router` package README

Paraphrased summary: `go_router` is Flutter's declarative URL-based routing package built on Router API. It supports path/query parameters, sub-routes, redirects, multiple Navigators through `ShellRoute`, Material/Cupertino apps, Navigator API compatibility, deep links, typed routes, state restoration, and migration guides. Pub.dev currently lists the package as 17.3.0, while this app pins `^14.0.0`, so local code must be checked before applying newer examples.
