# Kotlin Multiplatform docs attestation

1. Kotlin Multiplatform docs describe KMP as an open-source JetBrains technology for sharing code across Android, iOS, desktop, web, and server while retaining native development advantages.
2. Android Developers docs state Kotlin Multiplatform is officially supported by Google for sharing business logic between Android and iOS, and describe it as stable and production-ready.
3. KMP docs emphasize sharing business logic, including networking and data serialization, while allowing platform-specific source sets such as iOS targets.
