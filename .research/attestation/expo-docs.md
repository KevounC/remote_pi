# Expo docs attestation

1. Expo TypeScript docs state Expo has first-class support for TypeScript and that the JavaScript interface of the Expo SDK is written in TypeScript.
2. Expo's create-app docs describe `create-expo-app` as the CLI to create a React Native / Expo project with templates and boilerplate.
3. Expo Router docs describe Expo Router as a file-based routing library for React Native and web apps, available on Android, iOS, tvOS, and web.
4. Expo workflow docs frame Expo as a development workflow around React Native applications.
