# ConnectRPC attestation

1. Connect's site describes defining APIs with Protocol Buffers and generating idiomatic clients in Go, TypeScript, Swift, Kotlin, Dart, and Python.
2. Connect positions its API close to the `fetch` API and handles server-side routing, serialization, and compression.
3. The source found in this pass does not establish Rust first-class support from Connect itself; Rust support would require separate evaluation if choosing Connect for Remote Pi's Rust relay.
